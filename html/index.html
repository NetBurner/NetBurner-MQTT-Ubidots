HTML page "index.html" located in module flash memory

This page is located in on-chip flash. It will be displayed only when there is not an index.htm or index.html located in the external flash card. To verify this, you can use ftp and copy the sample flash card "index.html" in the project directory of this example.

To display a directory of the external flash card, use "DIR" in the URL. For example, http://10.1.1.2/DIR.

HTML pages used for dynamic web content that contain the FUNCTIONCALL HTML tags must be known at compile-time, and so must reside in the flash memory of the module. These pages can reference files on external flash cards.
